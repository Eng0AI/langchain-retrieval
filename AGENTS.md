# LangChain Retrieval

A LangChain.js retrieval chain with Supabase vector store for document Q&A, powered by the Vercel AI SDK.

## Tech Stack

- Next.js 15
- React 18
- LangChain.js
- Vercel AI SDK
- OpenAI
- Supabase (Vector Store)
- Tailwind CSS
